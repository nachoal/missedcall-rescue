Built to be turned on.

Not a flashy "autonomous company." A narrow operator a real business could switch on tomorrow.

Under the hood, the real Hermes agent runs on NVIDIA Nemotron, sandboxed by NVIDIA's NemoClaw / OpenShell with a deny-by-default network policy that blocks any off-policy call. The demo deposit runs through Stripe Checkout test mode, so it never touches raw card or bank details, and every agent spend is capped to vendors you pre-approved. Everything it does is written to a hash-chained, tamper-evident ledger you can audit line by line. You review the booked job and proof trail — not a pile of tasks.
